# bennettDISH.github.io

Bennett's personal **GitHub Pages** site — a hub of small static browser tools/experiments (audio site,
book reader, climbing scorer, connections creator, cooking site, database manager, JSON builder, lore
site, and more), served at `bennettdish.github.io`.

## Stack
Static HTML/CSS/JS · GitHub Pages (deploys on push to the default branch)

## Getting started
Open `index.html` in a browser, or `npx serve .` to preview locally.

## Layout
- `index.html`, `app.js`, `script.js`, `style.css` — hub landing.
- `pages/<tool>/index.html` — individual tools.
- `images/` — shared assets.
